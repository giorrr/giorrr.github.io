Artículo 1 – Condiciones de uso

La Aplicación se proporciona "TAL CUAL" (AS IS). El Proveedor no ofrece ninguna garantía en cuanto a la precisión, integridad, utilidad o idoneidad para un fin determinado de la Aplicación.

Artículo 2 – Funciones de la Aplicación

La Aplicación ofrece funcionalidad de alarma mediante la API AlarmKit de iOS y funcionalidad de restricción de uso de aplicaciones mediante la API Screen Time (FamilyControls / ManagedSettings / DeviceActivityMonitor).

  • La función de alarma activa una alarma a nivel del sistema a la hora configurada por el usuario.
  • La función de restricción de aplicaciones aplica temporalmente limitaciones de acceso a las aplicaciones preseleccionadas por el usuario.

Estas funciones operan en base a la configuración y los permisos otorgados por el propio usuario. El Proveedor no controla el dispositivo del usuario de forma remota.

Los usuarios no podrán reproducir, modificar, adaptar, descompilar, desensamblar, aplicar ingeniería inversa ni distribuir ninguna parte de la Aplicación sin autorización previa del Proveedor.

Artículo 4 – Actividades prohibidas

Los usuarios no deberán realizar las siguientes actividades al utilizar la Aplicación:
