Política de Privacidad

(Aplicación: Lock Alarm - Re: U)

Última actualización: 14 de mayo de 2026

Esta Política de Privacidad se aplica a la aplicación móvil Lock Alarm - Re: U ("la Aplicación") proporcionada por el operador de Lock Alarm - Re: U ("el Proveedor"). La Aplicación se proporciona "TAL CUAL" (AS IS).

1. Recopilación y uso de información

La Aplicación no requiere registro de usuario y no recopila directamente información de identificación personal. Sin embargo, el sistema operativo o las funciones estándar del sistema pueden recopilar automáticamente la siguiente información técnica limitada para garantizar el funcionamiento correcto y la mejora de la calidad:

  • Tipo de dispositivo
  • Versión del sistema operativo
  • Información sobre la estabilidad de la aplicación (registros de fallos, etc.)

La Aplicación no recopila datos de ubicación (incluida la ubicación precisa). La Aplicación no accede a la cámara, el micrófono, los contactos ni la biblioteca de fotos.

2. Uso de la API Screen Time

La Aplicación utiliza la API Screen Time de iOS (FamilyControls / ManagedSettings / DeviceActivityMonitor) para proporcionar funciones de restricción de uso de aplicaciones.

  • La información sobre las aplicaciones seleccionadas por el usuario se almacena en el dispositivo únicamente como tokens opacos (ApplicationToken). El Proveedor no puede ver ni obtener nombres de aplicaciones ni datos de uso.
  • Todos los datos adquiridos y procesados a través de la API Screen Time permanecen en el dispositivo del usuario y nunca se transmiten a servidores externos.
  • El uso de esta función requiere que el usuario otorgue permiso de Family Controls.

3. Almacenamiento de datos

La Aplicación no almacena datos del usuario en servidores externos. Todos los datos relacionados con la configuración de alarmas y las restricciones de uso de aplicaciones se almacenan en el dispositivo del usuario y son gestionados por el sistema operativo del dispositivo. Al desinstalar la Aplicación, los datos relacionados se eliminan de acuerdo con el comportamiento estándar del dispositivo.

4. Privacidad de los menores

La Aplicación no está destinada a personas menores de 13 años. El Proveedor no recopila intencionadamente información de identificación personal de menores de 13 años. Si se descubre que se ha recopilado información personal de un menor de 13 años, se tomarán las medidas oportunas de inmediato.

5. Seguridad

El Proveedor se esfuerza por proteger la información del usuario. La Aplicación utiliza los mecanismos de seguridad proporcionados por el sistema operativo para garantizar la seguridad de la información procesada en el dispositivo.

6. Cambios en esta política

Esta Política de Privacidad puede actualizarse según sea necesario. Los cambios se publicarán en esta página con una fecha de entrada en vigor actualizada. Se recomienda a los usuarios revisar esta Política de Privacidad periódicamente. El uso continuado de la Aplicación constituye la aceptación de la política actualizada.

7. Fecha de entrada en vigor

Esta Política de Privacidad entra en vigor el 14 de mayo de 2026.

8. Consentimiento

Al utilizar la Aplicación, usted acepta esta Política de Privacidad.
