1. Information Collection and Use

The App does not require user registration and does not directly collect any personally identifiable information. However, the following limited technical information may be automatically collected by the operating system or standard system features for the purpose of ensuring proper operation and quality improvement:

  • Device type
  • OS version
  • App stability information (crash logs, etc.)

The App does not collect location data (including precise location). The App does not access the camera, microphone, contacts, or photo library.

2. Use of Screen Time API

The App uses the iOS Screen Time API (FamilyControls / ManagedSettings / DeviceActivityMonitor) to provide app usage restriction features.

  • Information about apps selected by the user is stored on the device only as opaque tokens (ApplicationToken). The Provider cannot view or obtain app names or usage data.
  • All data acquired and processed through the Screen Time API remains on the user's device and is never transmitted to external servers.
  • Use of this feature requires the user to grant Family Controls permission.

3. Data Storage

The App does not store any user data on external servers. All data related to alarm settings and app usage restrictions is stored on the user's device and managed by the device's operating system. When the App is uninstalled, related data is deleted in accordance with the device's standard behavior.

4. Children's Privacy

The App is not intended for individuals under the age of 13. The Provider does not intentionally collect personally identifiable information from anyone under 13. If it comes to the Provider's attention that personal information has been collected from a child under 13, appropriate action will be taken promptly.

5. Security

The Provider strives to protect user information. The App uses security mechanisms provided by the operating system to ensure the safety of information processed on the device.
