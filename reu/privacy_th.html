นโยบายความเป็นส่วนตัว

(แอป: Lock Alarm - Re: U)

อัปเดตล่าสุด: 31 พฤษภาคม 2026

นโยบายความเป็นส่วนตัวนี้ใช้บังคับกับแอปพลิเคชันมือถือ Lock Alarm - Re: U ("แอป") ที่ให้บริการโดยผู้ดำเนินการ Lock Alarm - Re: U ("ผู้ให้บริการ") แอปนี้ให้บริการตามสภาพที่เป็นอยู่ (AS IS)

สารบัญ 1. การเก็บรวบรวมและการใช้ข้อมูล 2. การใช้ Screen Time API 3. บริการโฆษณา (Google AdMob) 4. สำหรับผู้ใช้ใน EU และ EEA (สิทธิ์ภายใต้ GDPR) 5. การจัดเก็บข้อมูล 6. ความเป็นส่วนตัวของเด็ก 7. ความปลอดภัย 8. การเปลี่ยนแปลงนโยบายนี้ 9. วันที่มีผลบังคับใช้ 10. การยินยอม 11. ติดต่อเรา

1. การเก็บรวบรวมและการใช้ข้อมูล

แอปไม่จำเป็นต้องให้ผู้ใช้ลงทะเบียน และไม่เก็บรวบรวมข้อมูลที่สามารถระบุตัวตนได้โดยตรง อย่างไรก็ตาม ข้อมูลทางเทคนิคที่จำกัดต่อไปนี้อาจถูกเก็บรวบรวมโดยอัตโนมัติโดยระบบปฏิบัติการหรือฟังก์ชันระบบมาตรฐาน เพื่อวัตถุประสงค์ในการทำให้แอปทำงานได้อย่างถูกต้องและปรับปรุงคุณภาพ:

  • ประเภทอุปกรณ์
  • เวอร์ชันของ OS
  • ข้อมูลเกี่ยวกับความเสถียรของแอป (เช่น บันทึกข้อผิดพลาด)

แอปไม่เก็บรวบรวมข้อมูลตำแหน่ง (รวมถึงข้อมูลตำแหน่งที่แม่นยำ) แอปไม่เข้าถึงกล้อง ไมโครโฟน รายชื่อผู้ติดต่อ หรือคลังรูปภาพ โปรดทราบว่าบริการโฆษณาของบุคคลที่สามอาจเก็บรวบรวมข้อมูลทางเทคนิคที่เกี่ยวข้องกับการแสดงโฆษณา

2. การใช้ Screen Time API

แอปใช้ iOS Screen Time API (FamilyControls / ManagedSettings / DeviceActivityMonitor) เพื่อให้บริการฟีเจอร์จำกัดการใช้งานแอป

  • ข้อมูลเกี่ยวกับแอปที่ผู้ใช้เลือกจะถูกจัดเก็บไว้ในอุปกรณ์เท่านั้นในรูปแบบโทเค็นที่ไม่ระบุข้อมูล (ApplicationToken) ผู้ให้บริการไม่สามารถดูหรือรับชื่อแอปหรือข้อมูลการใช้งานได้
  • ข้อมูลทั้งหมดที่ได้รับและประมวลผลผ่าน Screen Time API ยังคงอยู่ในอุปกรณ์ของผู้ใช้ และไม่เคยถูกส่งไปยังเซิร์ฟเวอร์ภายนอก
  • การใช้ฟีเจอร์นี้ต้องให้ผู้ใช้อนุญาตสิทธิ์ Family Controls

3. บริการโฆษณา (Google AdMob)

แอปใช้บริการโฆษณาของบุคคลที่สามเพื่อแสดงโฆษณา บุคคลที่สามเหล่านี้อาจเก็บรวบรวมข้อมูลทางเทคนิค เช่น ตัวระบุโฆษณา (IDFA) ที่อยู่ IP ข้อมูลอุปกรณ์ และข้อมูลการใช้งานแอป เพื่อแสดงโฆษณาที่ตรงกับความสนใจของผู้ใช้

บริการโฆษณาที่ใช้ในแอปรวมถึง Google AdMob สำหรับข้อมูลเกี่ยวกับวิธีที่ Google จัดการข้อมูล โปรดดูนโยบายต่อไปนี้:

  • นโยบายความเป็นส่วนตัวของ Google: https://policies.google.com/privacy
  • วิธีที่ Google ใช้ข้อมูลจากเว็บไซต์และแอปของพาร์ทเนอร์: https://policies.google.com/technologies/partner-sites

ผู้ให้บริการไม่สามารถควบคุมข้อมูลที่เก็บรวบรวมโดยบุคคลที่สามเหล่านี้ได้ อย่างไรก็ตาม แต่ละบริการจัดการข้อมูลตามนโยบายความเป็นส่วนตัวของตนเอง

4. สำหรับผู้ใช้ใน EU และ EEA (สิทธิ์ภายใต้ GDPR)

ส่วนนี้ใช้บังคับกับผู้ใช้ที่อาศัยอยู่ในสหภาพยุโรป (EU) และเขตเศรษฐกิจยุโรป (EEA)

ฐานทางกฎหมายในการประมวลผล
ข้อมูลส่วนบุคคลที่ประมวลผลเพื่อวัตถุประสงค์ในการโฆษณาในแอปอยู่บนพื้นฐานความยินยอมของคุณ (มาตรา 6(1)(a) ของ GDPR) แอปใช้ User Messaging Platform (UMP) ของ Google AdMob เพื่อขอความยินยอมเมื่อเปิดใช้งานครั้งแรกหรือตามที่จำเป็น

การโอนข้อมูลออกนอก EU/EEA
บริการโฆษณาของบุคคลที่สาม (รวมถึง Google AdMob) อาจโอนข้อมูลไปยังเซิร์ฟเวอร์นอก EU/EEA (โดยหลักอยู่ในสหรัฐอเมริกา) การโอนดังกล่าวดำเนินการภายใต้มาตรการคุ้มครองที่เหมาะสม รวมถึงข้อกำหนดสัญญามาตรฐาน (SCC / Standard Contractual Clauses) ที่ได้รับการอนุมัติจากคณะกรรมาธิการยุโรป

สิทธิ์ของคุณ (GDPR มาตรา 15–22)
หากคุณอาศัยอยู่ใน EU/EEA คุณมีสิทธิ์ต่อไปนี้:

  • สิทธิ์ในการเข้าถึง: ขอเข้าถึงข้อมูลส่วนบุคคลของคุณที่ประมวลผลโดยผู้ให้บริการหรือบุคคลที่สาม
  • สิทธิ์ในการแก้ไข: ขอแก้ไขข้อมูลส่วนบุคคลที่ไม่ถูกต้องหรือไม่ครบถ้วน
  • สิทธิ์ในการลบ ("สิทธิ์ที่จะถูกลืม"): ขอลบข้อมูลส่วนบุคคลของคุณภายใต้เงื่อนไขบางประการ
  • สิทธิ์ในการจำกัดการประมวลผล: ขอให้จำกัดการประมวลผลข้อมูลของคุณภายใต้เงื่อนไขบางประการ
  • สิทธิ์ในการโอนข้อมูล: รับข้อมูลที่คุณได้ให้ไว้ในรูปแบบที่มีโครงสร้างและอ่านได้ด้วยเครื่อง
  • สิทธิ์ในการคัดค้าน: คัดค้านการประมวลผลข้อมูลส่วนบุคคลของคุณเพื่อวัตถุประสงค์เฉพาะ

การปฏิเสธโฆษณาส่วนบุคคล
หากต้องการปฏิเสธโฆษณาแบบส่วนบุคคล (ที่กำหนดเป้าหมาย) คุณสามารถจำกัดการติดตามโฆษณาได้ในการตั้งค่าอุปกรณ์ของคุณ (iOS: การตั้งค่า > ความเป็นส่วนตัวและความปลอดภัย > การโฆษณาของ Apple > โฆษณาที่ปรับแต่งเฉพาะบุคคล)

การใช้สิทธิ์ของคุณ
หากต้องการใช้สิทธิ์ข้างต้น โปรดติดต่อเราผ่านแบบฟอร์มที่ด้านล่างของหน้านี้ สำหรับข้อมูลที่เก็บรวบรวมและประมวลผลโดย Google AdMob โปรดใช้หน้าคำขอลบข้อมูลของ Google

5. การจัดเก็บข้อมูล

แอปไม่จัดเก็บข้อมูลของผู้ใช้บนเซิร์ฟเวอร์ภายนอก ข้อมูลทั้งหมดที่เกี่ยวข้องกับการตั้งค่าการปลุกและการจำกัดการใช้งานแอปจะถูกจัดเก็บไว้ในอุปกรณ์ของผู้ใช้และจัดการโดยระบบปฏิบัติการของอุปกรณ์ เมื่อถอนการติดตั้งแอป ข้อมูลที่เกี่ยวข้องจะถูกลบออกตามพฤติกรรมมาตรฐานของอุปกรณ์

6. ความเป็นส่วนตัวของเด็ก

แอปไม่ได้มีไว้สำหรับบุคคลที่มีอายุต่ำกว่า 13 ปี ผู้ให้บริการไม่เก็บรวบรวมข้อมูลที่สามารถระบุตัวตนได้จากบุคคลที่มีอายุต่ำกว่า 13 ปีโดยเจตนา หากผู้ให้บริการทราบว่ามีการเก็บรวบรวมข้อมูลส่วนบุคคลจากเด็กที่มีอายุต่ำกว่า 13 ปี ผู้ให้บริการจะดำเนินการที่เหมาะสมโดยทันที

7. ความปลอดภัย

ผู้ให้บริการมุ่งมั่นในการปกป้องข้อมูลของผู้ใช้ แอปใช้กลไกความปลอดภัยที่ระบบปฏิบัติการจัดหาให้เพื่อให้มั่นใจในความปลอดภัยของข้อมูลที่ประมวลผลบนอุปกรณ์

8. การเปลี่ยนแปลงนโยบายนี้

นโยบายความเป็นส่วนตัวนี้อาจได้รับการอัปเดตตามความจำเป็น การเปลี่ยนแปลงจะถูกเผยแพร่บนหน้านี้พร้อมกับวันที่มีผลบังคับใช้ที่อัปเดต ผู้ใช้ควรตรวจสอบนโยบายความเป็นส่วนตัวนี้เป็นระยะ การใช้แอปต่อเนื่องถือเป็นการยอมรับนโยบายที่อัปเดตแล้ว

9. วันที่มีผลบังคับใช้

นโยบายความเป็นส่วนตัวนี้มีผลบังคับใช้ตั้งแต่วันที่ 31 พฤษภาคม 2026

10. การยินยอม

การใช้แอปถือว่าคุณยินยอมต่อนโยบายความเป็นส่วนตัวนี้ ผู้ใช้ที่อาศัยอยู่ใน EU/EEA สามารถถอนความยินยอมต่อการประมวลผลข้อมูลที่เกี่ยวข้องกับการโฆษณาได้ตลอดเวลา (GDPR มาตรา 7(3)) ความยินยอมสามารถถอนได้ผ่านการตั้งค่าโฆษณาของอุปกรณ์ของคุณ การถอนความยินยอมไม่มีผลต่อความชอบด้วยกฎหมายของการประมวลผลที่ดำเนินการก่อนการถอน
