1. Coleta e Uso de Informações

O Aplicativo não exige cadastro do usuário e não coleta diretamente nenhuma informação de identificação pessoal. No entanto, as seguintes informações técnicas limitadas podem ser coletadas automaticamente pelo sistema operacional ou por funções padrão do sistema, com o objetivo de garantir o funcionamento adequado e a melhoria da qualidade:

  • Tipo de dispositivo
  • Versão do OS
  • Informações de estabilidade do Aplicativo (logs de falhas, etc.)

O Aplicativo não coleta dados de localização (incluindo localização precisa). O Aplicativo não acessa a câmera, o microfone, os contatos nem a biblioteca de fotos. Observe que serviços de publicidade de terceiros podem coletar informações técnicas relacionadas à exibição de anúncios.

2. Uso da Screen Time API

O Aplicativo usa a Screen Time API do iOS (FamilyControls / ManagedSettings / DeviceActivityMonitor) para fornecer recursos de restrição de uso de aplicativos.

  • As informações sobre os aplicativos selecionados pelo usuário são armazenadas no dispositivo apenas como tokens opacos (ApplicationToken). O Fornecedor não consegue visualizar nem obter os nomes dos aplicativos ou os dados de uso.
  • Todos os dados obtidos e processados por meio da Screen Time API permanecem no dispositivo do usuário e nunca são transmitidos para servidores externos.
  • O uso desse recurso exige que o usuário conceda a permissão Family Controls.

3. Serviços de Publicidade (Google AdMob)

O Aplicativo usa serviços de publicidade de terceiros para exibir anúncios. Esses terceiros podem coletar informações técnicas como identificadores de publicidade (IDFA), endereços IP, informações do dispositivo e dados de uso do aplicativo, a fim de exibir anúncios adaptados aos interesses dos usuários.

Os serviços de publicidade utilizados no Aplicativo incluem o Google AdMob. Para informações sobre como o Google trata os dados, consulte as seguintes políticas:

  • Política de Privacidade do Google: https://policies.google.com/privacy
  • Como o Google usa dados de sites e aplicativos parceiros: https://policies.google.com/technologies/partner-sites

O Fornecedor não pode controlar as informações coletadas por esses terceiros; no entanto, cada serviço trata as informações de acordo com sua própria política de privacidade.

4. Para Usuários na UE e no EEE (Direitos sob o GDPR)

Esta seção aplica-se aos usuários residentes na União Europeia (UE) e no Espaço Econômico Europeu (EEE).

Base Legal para o Processamento
Os dados pessoais processados para fins de publicidade no Aplicativo baseiam-se no seu consentimento (Artigo 6(1)(a) do GDPR). O Aplicativo usa a User Messaging Platform (UMP) do Google AdMob para obter o consentimento na primeira execução ou conforme necessário.

Transferências de Dados para Fora da UE/EEE
Os serviços de publicidade de terceiros (incluindo o Google AdMob) podem transferir dados para servidores fora da UE/EEE (principalmente nos Estados Unidos). Tais transferências são realizadas sob salvaguardas adequadas, incluindo as Cláusulas Contratuais Padrão (SCC) aprovadas pela Comissão Europeia.

Seus Direitos (Artigos 15 a 22 do GDPR)
Se você reside na UE/EEE, tem os seguintes direitos:

  • Direito de acesso: Solicitar acesso aos seus dados pessoais processados pelo Fornecedor ou por terceiros.
  • Direito de retificação: Solicitar a correção de dados pessoais incorretos ou incompletos.
  • Direito ao apagamento ("direito ao esquecimento"): Solicitar a exclusão dos seus dados pessoais sob determinadas condições.
  • Direito à limitação do processamento: Solicitar que o processamento dos seus dados seja limitado sob determinadas condições.
  • Direito à portabilidade dos dados: Receber os dados que você forneceu em um formato estruturado e legível por máquina.
  • Direito de oposição: Opor-se ao processamento dos seus dados pessoais para fins específicos.

Recusar Anúncios Personalizados
Para recusar a publicidade personalizada (direcionada), você pode limitar o rastreamento de anúncios nas configurações do seu dispositivo (iOS: Configurações > Privacidade e Segurança > Publicidade da Apple > Anúncios Personalizados).

Exercício dos Seus Direitos
Para exercer os direitos acima, entre em contato conosco por meio do formulário no final desta página. Para os dados coletados e processados pelo Google AdMob, use a página de solicitação de exclusão de dados do Google.

5. Armazenamento de Dados

O Aplicativo não armazena nenhum dado do usuário em servidores externos. Todos os dados relacionados às configurações de alarme e às restrições de uso de aplicativos são armazenados no dispositivo do usuário e gerenciados pelo sistema operacional do dispositivo. Quando o Aplicativo é desinstalado, os dados associados são excluídos de acordo com o comportamento padrão do dispositivo.

6. Privacidade de Crianças

O Aplicativo não é destinado a pessoas menores de 13 anos. O Fornecedor não coleta intencionalmente informações de identificação pessoal de pessoas menores de 13 anos. Caso o Fornecedor tome conhecimento de que informações pessoais foram coletadas de uma criança menor de 13 anos, medidas adequadas serão tomadas imediatamente.

7. Segurança

O Fornecedor se esforça para proteger as informações dos usuários. O Aplicativo usa os mecanismos de segurança fornecidos pelo sistema operacional para garantir a proteção das informações processadas no dispositivo.

8. Alterações nesta Política

Esta Política de Privacidade pode ser atualizada conforme necessário. As alterações serão publicadas nesta página com uma data de vigência atualizada. Recomendamos que os usuários revisem esta Política de Privacidade periodicamente. O uso continuado do Aplicativo constitui a aceitação da política atualizada.

9. Data de Vigência

Esta Política de Privacidade está em vigor desde 31 de maio de 2026.

10. Consentimento

Ao usar o Aplicativo, você consente com esta Política de Privacidade. Os usuários residentes na UE/EEE podem retirar seu consentimento ao processamento de dados relacionado à publicidade a qualquer momento (Artigo 7(3) do GDPR). O consentimento pode ser retirado por meio das configurações de publicidade do seu dispositivo. A retirada do consentimento não afeta a licitude do processamento realizado antes da retirada.
