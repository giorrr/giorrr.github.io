1. 情報の取得および利用について

本アプリは、ユーザー登録を必要とせず、ユーザーから個人を特定できる情報を直接取得することはありません。ただし、本アプリの正常な動作および品質向上のため、オペレーティングシステムや標準的なシステム機能により、以下のような限定的な技術情報が自動的に取得される場合があります。

  • 端末の種類
  • OSのバージョン
  • アプリの安定性に関する利用情報（クラッシュログ等）

本アプリは、位置情報（正確な位置情報を含む）を取得しません。また、カメラ、マイク、連絡先、写真ライブラリへのアクセスを行いません。なお、広告表示に関連して、第三者サービスが技術的情報を取得する場合があります。

2. Screen Time API の利用について

本アプリは、アプリ使用制限機能を提供するために、iOSのScreen Time API（FamilyControls / ManagedSettings / DeviceActivityMonitor）を使用します。

  • ユーザーが選択したアプリの情報は、不透明なトークン（ApplicationToken）としてのみ端末内に保存されます。提供者がアプリ名やアプリの利用状況を閲覧・取得することはできません。
  • Screen Time APIを通じて取得・処理されるデータは、すべてユーザーの端末内に留まり、外部サーバーへ送信されることはありません。
  • この機能の利用には、ユーザーによるFamily Controls権限の許可が必要です。

3. 広告配信サービスについて（Google AdMob）

本アプリでは、広告配信を目的として、第三者の広告配信サービスを利用しています。これらの第三者は、ユーザーの興味に応じた広告を表示するため、広告識別子（IDFA）、IPアドレス、端末情報、アプリ使用情報等の技術的情報を収集することがあります。

本アプリで使用する広告配信サービスには、Google AdMob が含まれます。Googleによる情報の取り扱いについては、以下のポリシーをご確認ください。

  • Googleプライバシーポリシー：https://policies.google.com/privacy
  • Googleパートナーサイトおよびアプリでのデータの利用：https://policies.google.com/technologies/partner-sites

提供者は、これら第三者が取得する情報を管理することはできませんが、各サービスはそれぞれのプライバシーポリシーに基づいて情報を取り扱います。

4. EUおよびEEA圏のユーザーへ（GDPRに基づく権利）

本セクションは、欧州連合（EU）および欧州経済領域（EEA）に居住するユーザーに適用されます。

個人データ処理の法的根拠
本アプリにおける広告配信を目的とした個人データの処理は、ユーザーの同意（GDPR第6条第1項(a)）に基づきます。本アプリは、Google AdMobが提供する同意管理プラットフォーム（UMP: User Messaging Platform）を通じて、初回起動時または必要に応じて同意を取得します。

EUおよびEEA域外へのデータ移転
第三者広告配信サービス（Google AdMob）は、データをEU/EEA域外のサーバー（主に米国）へ転送する場合があります。かかる転送は、欧州委員会が承認した標準契約条項（SCC）等の適切な保護措置に基づいて行われます。

お客様の権利（GDPR第15条〜第22条）
EU/EEAに居住するユーザーは、以下の権利を有します。

  • アクセス権：提供者または第三者が処理するあなたの個人データへのアクセスを要求する権利
  • 訂正権：不正確または不完全な個人データの訂正を要求する権利
  • 削除権（忘れられる権利）：一定の条件のもと、個人データの削除を要求する権利
  • 処理制限権：一定の条件のもと、個人データの処理を制限することを要求する権利
  • データポータビリティ権：提供されたデータを構造化された機械可読形式で受け取る権利
  • 異議申し立て権：特定の目的のための個人データ処理に異議を申し立てる権利

パーソナライズド広告のオプトアウト
パーソナライズド広告（ターゲティング広告）を希望しない場合は、お使いの端末の設定から広告トラッキングを制限できます（iOS：設定 > プライバシーとセキュリティ > Apple広告 > パーソナライズされた広告）。

権利行使のお問い合わせ
上記の権利行使に関するご要望は、本ページ末尾のお問い合わせフォームよりご連絡ください。Google AdMobが収集・処理するデータについては、Googleのデータ削除リクエストページよりご対応ください。

5. データの保存について

本アプリは、ユーザーの個人データを外部サーバーに保存しません。アラーム設定およびアプリ使用制限に関するデータはすべてユーザーの端末内に保持され、管理は端末のOSに委ねられます。本アプリをアンインストールした場合、関連するデータは端末の標準的な動作に従って削除されます。

6. お子様のプライバシーについて

本アプリは、13歳未満の方を対象としていません。提供者は、13歳未満の方から個人を特定できる情報を意図的に取得することはありません。万が一、13歳未満の方の個人情報が取得されたことが判明した場合、速やかに適切な対応を行います。

7. セキュリティについて

提供者は、ユーザー情報の保護に努めています。本アプリでは、OSが提供するセキュリティ機構を利用し、端末内で処理される情報の安全性を確保します。

8. プライバシーポリシーの変更

本プライバシーポリシーは、必要に応じて変更されることがあります。変更後の内容は、本ページに掲載し、効力発生日を更新します。ユーザーは、本プライバシーポリシーを定期的にご確認ください。本アプリの継続利用をもって、変更後の内容に同意したものとみなされます。

9. 効力発生日

10. 同意について

ユーザーは、本アプリを利用することで、本プライバシーポリシーに同意したものとします。EU/EEAに居住するユーザーは、広告配信に関する同意をいつでも撤回することができます（GDPR第7条第3項）。同意の撤回は、端末の広告設定から行うことができます。同意の撤回は、撤回前に行われた処理の適法性には影響しません。
