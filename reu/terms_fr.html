L'Application fournit des fonctionnalités d'alarme à l'aide de l'AlarmKit API d'iOS ainsi que des fonctionnalités de restriction de l'utilisation des applications à l'aide de la Screen Time API (FamilyControls / ManagedSettings / DeviceActivityMonitor).
