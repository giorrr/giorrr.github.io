Articolo 2 – Funzionalità dell'App

L'App fornisce funzionalità di sveglia tramite l'AlarmKit API di iOS e funzionalità di limitazione dell'utilizzo delle app tramite la Screen Time API (FamilyControls / ManagedSettings / DeviceActivityMonitor).

  • La funzione di sveglia attiva una sveglia a livello di sistema all'orario impostato dall'utente.

Queste funzionalità operano in base alle impostazioni e alle autorizzazioni dell'utente stesso. Il Fornitore non controlla da remoto il dispositivo dell'utente.
